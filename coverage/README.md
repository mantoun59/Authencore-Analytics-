# Test Coverage Report

This directory contains test coverage reports generated by Vitest.

## Viewing Coverage

1. Run tests with coverage:
   ```bash
   npm run test:coverage
   ```

2. Open the HTML report:
   ```bash
   open coverage/index.html
   ```

## Coverage Goals

- **Statements**: 80%
- **Branches**: 75%
- **Functions**: 80%
- **Lines**: 80%
